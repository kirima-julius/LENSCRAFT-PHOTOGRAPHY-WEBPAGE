Photography Website Design

Before developing this website, I created a simple layout showing the structure of the pages and the placement of the major components. This design served as the blueprint throughout the development process.

Design Objectives

  • Create a clean and simple navigation menu.
  • Present photography collections in an organized gallery.
  • Provide an About Me section introducing the photographer.
  • Include an easy-to-use contact form.
  • Maintain a consistent colour palette throughout the website.
  • Ensure the layout is responsive on different screen sizes.

Development Process

  1. Designed the website layout.
  2. Created the HTML page structure.
  3. Added photographs and website content.
  4. Styled the website using CSS.
  5. Tested responsiveness on different screen sizes.
  6. Published the project using GitHub Pages.
